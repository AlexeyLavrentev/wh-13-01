Это типовая ОС для экспериментов в области информационной безопасности, с которой следует начать при анализе уязвимостей.

Для этого нужно в поиске ввести название сетевой службы, обнаруженной на атакуемой машине, и выбрать подходящие по версии уязвимости.

## Задание 2
Проведите сканирование Metasploitable в режимах SYN, FIN, Xmas, UDP.
Запишите сеансы сканирования в Wireshark.
Ответьте на следующие вопросы:
Чем отличаются эти режимы сканирования с точки зрения сетевого трафика?
Как отвечает сервер?
Приведите ответ в свободной форме.

## Ответ:

При сканировании разных типов были замечены различия в сетевом трафике.
SYN-сканирование посылает пакет SYN и ждет ответ SYN/ACK. Если порт открыт, сервер отвечает SYN/ACK, после чего сканер отправляет RST, и соединение не завершается.
FIN-сканирование отправляет пакет с флагом FIN. Если порт закрыт, сервер отвечает RST. Открытый порт, согласно RFC 793, не должен отвечать, поэтому ответа нет.
Xmas-сканирование отправляет TCP-пакет с установленными флагами FIN, PSH и URG. Сервер ведет себя так же, как при FIN-сканировании: закрытый порт отвечает RST, на открытый порт ответа нет.
UDP-сканирование посылает UDP-пакеты. Если порт закрыт, сервер возвращает ICMP-сообщение «Port Unreachable». Открытый порт может ответить или не ответить.
Таким образом, разные типы сканирования различаются используемыми TCP/UDP-пакетами и реакцией сервера на них.
